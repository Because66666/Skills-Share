# 用户操作手册

本手册旨在帮助不同角色的用户快速掌握智慧法务系统的核心功能操作。

## 1. 合同管理

### 1.1 发起合同审查
1. 点击导航栏“合同管理” -> “合同审查”。
2. 点击右上角“发起审查”按钮。
3. 上传合同文件（支持 Word/PDF），选择合同类型。
4. 系统将自动进行 AI 预审，并生成审查报告。

### 1.2 合同归档
*   **自动归档**：审批通过后的合同将自动进入归档库。
*   **手动归档**：对于历史合同，支持批量导入归档。

## 2. 案件中心

### 2.1 案件登记
在“案件管理”模块，点击“新增案件”，填写案件基本信息（案号、当事人、涉案金额等）并指派承办律师。

### 2.2 进展更新
承办律师需在案件详情页的“案件进展”标签下，实时更新庭审记录、证据上传等信息。

## 3. 知识产权

### 3.1 商标监控
系统每日自动同步商标局数据，对企业名下商标进行状态监控。如有异常（如被抢注、即将到期），系统会发送告警通知。

## 4. 风险监控

系统接入工商、司法大数据，对企业的关联方、合作伙伴进行实时风险扫描。
*   **红灯预警**：重大失信、被执行。
*   **黄灯预警**：股权冻结、行政处罚。
