# 快速入门

欢迎使用智慧法务系统！本指南将帮助您快速上手系统的基本操作。

## 1. 登录系统

访问系统地址，输入您的账号和密码。如果您是首次登录，建议先修改初始密码。

## 2. 首页仪表盘

登录成功后，您将看到仪表盘（Dashboard）。这里展示了您需要关注的核心指标和待办事项。
*   **待办事项**：点击可以直接跳转处理。
*   **数据图表**：将鼠标悬停可查看详细数据。

## 3. 常用操作

### 发起合同审查
1.  进入“合同管理”模块。
2.  点击“发起合同”按钮。
3.  上传合同文件，填写基本信息。
4.  系统将自动进行AI预审，随后流转至法务人员。

### 查询案件
1.  进入“案件管理”模块。
2.  在搜索框输入案号或当事人名称。
3.  点击列表项查看案件详情。

## 4. 获取帮助

如果在操作过程中遇到问题，您可以：
*   查阅本“文档中心”。
*   访问“帮助中心”搜索常见问题。
*   联系系统管理员。
