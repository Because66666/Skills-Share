# 因子分析综合评价使用示例

## 示例1：学生成绩综合评价
### 输入数据
假设我们有5名学生的4门科目成绩：
```python
import pandas as pd

data = pd.DataFrame({
    '语文': [85, 78, 92, 70, 80],
    '数学': [90, 82, 88, 65, 85],
    '英语': [88, 80, 95, 72, 83],
    '物理': [92, 75, 90, 68, 87]
}, index=['学生A', '学生B', '学生C', '学生D', '学生E'])
```

### 调用方法
```python
from scripts.model import factor_analysis_comprehensive_evaluation

results = factor_analysis_comprehensive_evaluation(data)
```

### 预期输出
1. **适用性检验**：
   - KMO检验统计量约为0.82（适合因子分析）
   - 巴特利特球形检验P值<0.05（拒绝原假设，数据适合因子分析）

2. **因子提取**：自动提取2个公共因子
   - 因子1载荷在语文、英语上较高（语言类因子）
   - 因子2载荷在数学、物理上较高（理科类因子）

3. **综合得分与排名**：
   | 学生   | 综合得分 | 排名 |
   |--------|----------|------|
   | 学生C  | 1.52     | 1    |
   | 学生A  | 0.87     | 2    |
   | 学生E  | 0.23     | 3    |
   | 学生B  | -0.61    | 4    |
   | 学生D  | -2.01    | 5    |
